Random Penrose Pattern Generator

What is that?

Penrose tiling is a never repeating infinity pattern which becomed his name after Roger Penrose (mathematician and physicist) in the 1970s. For that aperiodic pattern you need just 2 different certain shapes, and follow some placement rules. For more information about what exactly a Penrose pattern is, please visit Wikipedia.

I think the most known tiles are romboids, but for that project I used an other variants, the Kite and Dart shapes. This generator generates an accurate random pattern from these. You have the possibility to adjust some properties of the tiling before generate, so feel free to play aroud.

Why I do that?

I wanted for an other Project a dinamically changing Penrose pattern background, but I nowhere could found a code or a random generator for that (in best case an API). I do learn Programming and I belive, the best way to develop skills is just coding line by line and use what you have learned, so I decided make it on my own. The real goal for that project is to make an API too for the pattern generator, so that everyone can easy implement it in his work when will a dinamically changing design.

It's made for us!

You can use it for generating an impressive backgrounds, printouts and for your other design works. Feel free to use, copy, paste or modify the generated image and the whole code in your projects.
